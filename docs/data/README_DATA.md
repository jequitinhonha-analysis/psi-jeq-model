# PSI_YÉKIT 1.0 — Dataset (anonimizados/agregados)

## Visão geral
Este pacote contém dados **anonimizados e/ou agregados** utilizados pelo PSI_YÉKIT 1.0 (Ψ_YÉKIT) para estimativa do estado latente de risco (Ψ) e validação preditiva.

## Estrutura do pacote
- data/processed/            Dados processados e anonimizados/agregados
- DATA_DICTIONARY.csv        Dicionário de variáveis (nome, tipo, unidade, descrição)
- PROVENANCE.md              Fontes, termos de uso e transformações principais
- ANONYMIZATION.md           Procedimento de anonimização/agregação e justificativas
- CHECKSUMS.txt              Hashes para verificação de integridade
